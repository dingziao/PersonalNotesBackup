Faiss

是一个向量搜索库。不是服务。不基于ML。

不是DB，不支持持久化存储。建立的Index文件可以导出到本地，避免重复建Index。

Faiss 和 K-NearestNeighbors 生态位相同。


区分作为向量距离计算的K-NearestNeighbors，和作为ML算法的KNN

KNN 作为ML算法：利用K-Nearest Neighbors 的标签，预测输入值的标签。

K-NearestNeighbors 本身不基于ML，只计算向量距离。

  • OpenAI API + Faiss 实现RAG示例：
    import faiss
import openai

# 建索引
index = faiss.IndexFlatL2(embs.shape[1])
index.add(embs)

# 查询embedding
q_emb = np.array(openai.Embedding.create(input="问题", model="text-embedding-3-small")["data"][0]["embedding"], dtype='float32')

# 检索
_, ids = index.search(q_emb.reshape(1,-1), k=1)
ctx = texts[ids[0][0]]

# 调用GPT
resp = openai.ChatCompletion.create(model="gpt-4o", messages=[{"role":"user","content":ctx + "\n回答问题：问题"}])
