Post-training：

  • RLHF属于Post-training。

推理 Reasoning Ability：

大多数现代LLM都具备基本的推理能力。

如今当我们提到推理时，通常指需要复杂、多步生成并包含中间步骤的问题。

CoT和RLHF都是提升推理能力的方式。

  1. CoT:
    • Chain-of-Thought(CoT)是用prompt鼓励模型推理。
    • CoT 是一种Prompt Engineering 技巧，通过给模型示范或者暗示让它输出推理过程。
    • 可以提升复杂推理题的表现。
  1. RLHF:
    • SFT（Supervised Fine-Tuning）：用人类问答微调模型
    • RL（Reinforcement Learning）：使用Reward Model多轮交互训练
    • RLHF（Reinforcement Learning with Human Feedback）分三步：
      • SFT
      • HF训练一个 Reward Model
      • 使用3.2 Reward Model 通过 RL 调优LLM

Prompt:

Zero-shot prompt：语言命令，不给例子（Step by step）
Few-shot ReAct：给出一些例子（Q&A Example）

Agent:

ReAct（Reasoning + Acting）是一种架构思想，不是具体的包。

Loop 控制器，Prompt 模板，工具路由器，环境控制器(Memory)

Action: []

Final Answer:
  • Example
    You are an AI agent that can think and act.

Available actions:  
- calc_square: calculates the square of a single number.  
- calc_add: calculates the sum of two numbers.

Use the following format:

Question: the input question you must answer

Thought: you should always think about what to do, plan your next step clearly.  
Action: the action to take, must be one of [calc_square, calc_add]  
Action Input: the input parameters for the action (single number for calc_square, two numbers separated by comma for calc_add)  
Observation: the result of the action  
... (repeat Thought/Action/Action Input/Observation as needed)

Final Answer: the answer to the original question
